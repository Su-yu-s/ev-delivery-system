Core Modules

Everything you need,
in one platform.

◆

Intelligent Route Planning

Powered by Amap API — input start and end addresses to instantly calculate optimal routes with distance, estimated time, and energy consumption. Real-time traffic-aware pathfinding keeps every delivery on schedule.

●

AI Copilot

DeepSeek-powered chatbot assists with route optimization, weather checks, and delivery strategy — all from a collapsible sidebar.

◇

Charging Network

Discover nearby charging stations mapped along your route. Color-coded markers at 1/4, midpoint, and 3/4 distances ensure you never run low.

▨

Role-Based Access

Fine-grained permission system with Admin and User roles. Four dedicated RBAC tables ensure every API endpoint is properly gated.

▶

Full Task Lifecycle Management

Create, track, and complete delivery tasks with a clean status pipeline — Pending, In-Progress, and Completed. Filterable task table with pagination keeps operations organized at any scale.

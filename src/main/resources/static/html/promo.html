面面俱到。

智能路线规划

基于高德地图 API，输入起终点即可秒级计算最优配送路线，同步输出里程、预估耗时与电量消耗。实时路况感知，让每一单都准时到达。

AI 智能副驾

DeepSeek 大模型驱动，可折叠侧边栏对话面板。支持路线优化建议、天气查询、充电策略——让 AI 成为你的配送搭档。

充电桩网络

沿途充电站自动标注，按 1/4 路程、中点、3/4 路程分级着色，续航焦虑一扫而空。

角色权限体系

管理员与普通用户双角色，四张 RBAC 数据表精细控制每个 API 端点。JWT 认证 + Spring Security 护航。

任务全生命周期管理

创建、追踪、完成——清晰的待配送 → 配送中 → 已完成状态流转。可筛选任务列表配合分页组件，从容应对任意规模配送业务。

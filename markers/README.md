# 充电站图标文件夹

此文件夹用于存放充电站标记的自定义图标。

## 当前图标需求
- 红色图标：起点充电站
- 蓝色图标：1/4路程充电站
- 黄色图标：中点充电站
- 绿色图标：3/4路程充电站
- 紫色图标：终点充电站

## 图标规格要求
- 格式：PNG（支持透明背景）
- 尺寸：30x40像素
- 命名规范：
  - start.png - 起点图标
  - quarter1.png - 1/4路程图标
  - midpoint.png - 中点图标
  - quarter3.png - 3/4路程图标
  - end.png - 终点图标

## 使用方法
1. 将自定义图标文件放入此文件夹
2. 在script.js中修改图标路径为本地路径
3. 确保图标文件名与代码中的引用一致
